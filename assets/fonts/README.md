Font files are not bundled in this package.

CSS has been prepared for these optional local files:
- JimmyScript-Rg.otf
- CocoGothic-Light_trial.ttf
- CocoGothic-Heavy_trial.ttf
- Bimbo Trial.ttf

The title, hashtag, parents, and full-name lockups are PNG artwork rendered from the uploaded trial fonts, so the design remains stable without bundling font files.

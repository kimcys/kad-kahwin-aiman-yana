Font files are not bundled in this package.

The site can render without them because the decorative title and hashtag are PNG artwork, and the CSS falls back to installed system/web fonts.

If you want the exact trial fonts locally, install them on your machine or wire up your own `@font-face` paths.

Common trial font names used by this design:
- JimmyScript-Rg.otf
- CocoGothic-Light_trial.ttf
- CocoGothic-Heavy_trial.ttf
- Bimbo Trial.ttf
